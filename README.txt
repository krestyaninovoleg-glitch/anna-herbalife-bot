PRO Telegram bot files for GitHub

Files:
- bot.py
- requirements.txt

Environment variables to set on Railway/Render:
- BOT_TOKEN=your_telegram_bot_token
- ADMIN_CHAT_ID=your_telegram_user_id

Quick start:
1. Upload bot.py and requirements.txt to GitHub
2. Connect repo to Railway
3. Add BOT_TOKEN and ADMIN_CHAT_ID
4. Deploy
